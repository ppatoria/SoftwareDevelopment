Spring and Hibernate Classes

UML Package Diagram Example

An example of UML package diagram representing some Spring and Hibernate data access classes.

The Spring Framework provides integration with several Object Relational Mapping (ORM) data access technologies including Hibernate, JDO, Oracle TopLink, iBATIS SQL Maps and JPA. Support packages for object relational mappers comply with Spring's generic transaction and DAO exception hierarchies. Spring framework hibernate3 packages use several Hibernate packages.
